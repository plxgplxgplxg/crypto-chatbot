# Crypto Chatbot

Frontend React + TypeScript (Vite) + TailwindCSS.

## Prerequisites
- Node.js 16+ (or LTS)
- npm, yarn or pnpm

## Install
```bash
# npm
npm install
# or
npm i
# yarn
yarn
# pnpm
pnpm install
```

## Run (development)
```bash
# npm
npm run dev
# yarn
yarn dev
# pnpm
pnpm dev
```

## Build
```bash
# npm
npm run build
```

## Preview production build
```bash
# npm
npm run preview
```

## Notes
- TailwindCSS and Vite are preconfigured. Nếu cần thay đổi cổng, chỉnh `vite.config.ts` hoặc export `PORT` trước khi chạy.
- Không cần API keys để chạy frontend cục bộ trừ khi project sử dụng backend riêng.
